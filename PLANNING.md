# Statepatmal 앱 기획서

## 📱 앱 개요

**앱 이름**: Statepatmal  
**부제**: 연인·부부·가족 간 "말 안 해도 되는 관리"  
**컨셉**: 관계 관리용 소프트 체크인 앱

---

## 🎯 문제 정의

### 핵심 문제
- 관계 문제의 **70%는 기대치 불일치**에서 발생
- 직접 말하면 **감정 소모**가 큼
- 매일 "어떻게 지내?" 같은 질문에 답하기 부담스러움
- 상대방의 상태를 알고 싶지만 물어보기 어려운 상황

### 해결 방안
**"말 안 해도 알게 해주는"** 자동화된 상태 공유 시스템

---

## ✨ 핵심 기능

### 1. 하루 1문장 상태 체크
- **형식**: 이모지 + 짧은 문장 (최대 50자)
- **예시**:
  - 😌 "오늘은 혼자 있고 싶음"
  - 💚 "지금 연락하면 좋음"
  - 😊 "오늘 기분 좋음"
  - 😔 "조금 힘듦"
  - 🎉 "오늘 뭔가 기대됨"
- **시간**: 하루 1회만 입력 가능 (자정 리셋)
- **UI**: 간단한 입력창 + 이모지 선택기

### 2. 요약 상태 전달
- **본인**: 전체 문장 확인 가능
- **상대방**: 요약된 상태만 표시
  - 이모지 + 감정 카테고리 (예: "혼자 있고 싶음", "연락 가능", "기분 좋음")
  - 상세 문장은 선택적으로 공개 가능
- **프라이버시**: 개인 설정으로 공개 범위 조절

### 3. 감정 누적 그래프
- **주간/월간 감정 트렌드** 시각화
- **이모지별 빈도** 분석
- **관계 패턴** 인사이트 (예: "이번 주는 많이 힘들었네요")
- **비교 기능**: 상대방과의 감정 동기화 정도 확인

## 🏗️ 앱 구조

### 화면 구성

#### 1. **온보딩/로그인 화면**
- 앱 소개 (3-4페이지)
- 회원가입/로그인
- 관계 설정 (커플/가족/팀)

#### 2. **홈 화면 (메인)**
- **내 상태 입력 카드**
  - 오늘 상태 입력 (이모지 + 문장)
  - 입력 여부 표시
- **상대방 상태 카드**
  - 요약 상태 표시
  - 마지막 업데이트 시간
- **감정 그래프 미리보기**
  - 최근 7일 트렌드

#### 3. **상태 입력 화면**
- 이모지 선택 (카테고리별)
- 텍스트 입력 (50자 제한)
- 자동 저장

#### 4. **감정 분석 화면**
- 주간/월간 그래프
- 이모지별 통계
- 관계 인사이트
- 상대방과 비교 (옵션)

#### 5. **설정 화면**
- 프로필 관리
- 관계 관리 (상대방 추가/제거)
- 알림 설정
- 프라이버시 설정
- 데이터 내보내기

## 🔄 사용자 플로우

### 첫 사용자
1. 앱 설치 → 온보딩
2. 회원가입
3. 관계 타입 선택 (커플/가족/팀)
4. 상대방 초대 (링크/코드)
5. 첫 상태 입력

### 일일 사용자
1. 앱 실행
2. 홈 화면에서 오늘 상태 확인
3. 상태 입력 (30초 이내)
4. 상대방 상태 확인
5. 필요시 그래프 확인

## 📊 데이터 모델

### 사용자 (User)
```dart
- id: String
- name: String
- email: String
- createdAt: DateTime
```

### 관계 (Relationship)
```dart
- id: String
- type: RelationshipType (couple, family, team)
- members: List<User>
- createdAt: DateTime
```

### 상태 (Status)
```dart
- id: String
- userId: String
- relationshipId: String
- emoji: String
- message: String (최대 50자)
- category: EmotionCategory
- isPublic: bool
- createdAt: DateTime
```

### 감정 카테고리 (EmotionCategory)
- 혼자 있고 싶음
- 연락 가능
- 기분 좋음
- 힘듦
- 기대됨
- 평온함
- 등등...

## 🎨 UI/UX 디자인 컨셉

### 디자인 원칙
- **미니멀**: 불필요한 요소 제거
- **밝은 보라색**: 보라빛나는 밝고 따뜻한 느낌
- **부드러움**: 감정을 다루는 앱이므로 부드러운 색상과 곡선
- **직관적**: 3초 안에 이해 가능
- **프라이버시 중심**: 개인 정보 보호 강조

### 색상 팔레트
- **메인**: 밝은 보라색 (Lavender, Light Purple)
  - Primary: #9C88FF (밝은 보라)
  - Secondary: #B794F6 (연한 보라)
  - Accent: #E8D5FF (아주 연한 보라)
- **배경**: 밝은 화이트/크림톤 (#FFFBFE, #F8F4FF)
- **그라데이션**: 보라색 그라데이션 효과 활용
- **강조**: 감정별 색상 (기쁨=노랑, 평온=초록, 힘듦=파랑 등)을 보라색 톤과 조화롭게

### 타이포그래피
- 본문: 읽기 쉬운 sans-serif (Pretendard, Noto Sans 등)
- 이모지: 큰 사이즈로 강조 (최소 48px)
- 제목: 보라색 계열로 강조

### 디자인 요소
- **카드**: 부드러운 그림자와 둥근 모서리 (borderRadius: 20-24px)
- **버튼**: 그라데이션 배경 또는 밝은 보라색 배경
- **아이콘**: 보라색 계열로 통일
- **애니메이션**: 부드러운 페이드인/아웃 효과
